# Fiber One Asia Website Rebuild

## 1. Project Description
Rebuild of the Fiber One Asia corporate website — an AI implements AV solutions provider bridging cutting-edge hardware engineering with intelligent software. Based in Singapore with regional offices across Asia, the company designs, builds, and deploys AV systems powered by machine learning, computer vision, and predictive analytics. The website showcases their AI-powered solutions, company profile, and provides a contact channel for potential clients.

## 2. Page Structure
- `/` - Home (hero slider, company info, solutions, contact form, footer)
- `/about` - Company Profile
- `/testimonials` - Client Testimonials
- `/careers` - Careers
- `/solutions` - Solutions Details
- `/gallery` - Project Gallery
- `/av-book` - Download AV Book
- `/contact` - Contact Page
- `/shop` - Shop (optional, based on user decision)
- `/privacy-policy` - Privacy Policy

## 3. Core Features
- [x] Hero slider with 9 AI-branded solution slides
- [x] Company information section emphasizing AI + hardware + software
- [x] Solutions carousel with AI-powered descriptions
- [x] Contact form with submission
- [x] Responsive navigation with mobile menu
- [x] Footer with branding
